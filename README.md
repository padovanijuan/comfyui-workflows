# ComfyUI Workflows 🎨

AI image and video generation workflows using ComfyUI with 
custom-trained LoRA models for consistent character generation at scale.

## Overview

A collection of AI generation workflows built for brand-consistent 
content creation. Uses custom LoRA models trained on specific character 
references to maintain visual identity across hundreds of generated 
assets, eliminating the need to re-describe characters in every prompt.

## Workflows

### Image Generation
- Custom LoRA model integration for consistent character identity
- Structured prompt templates controlling style, lighting and composition
- Reusable prompt frameworks for brand-aligned visual content

### Video Generation
- AI avatar technology with voice synthesis
- Product promotion content for TikTok Shop
- Automated batch generation pipeline

## Tech Stack

- **Generation** — ComfyUI
- **Models** — Custom LoRA (character consistency)
- **Cloud** — RunPod (high-performance GPU)
- **Video** — AI Avatar + Voice Synthesis
- **Distribution** — Replicate

## Key Principles

- Brand consistency through LoRA training — not prompt repetition
- Structured prompt frameworks for repeatable results
- Cloud-based generation for scale and performance
- Iterative model training based on output quality feedback
